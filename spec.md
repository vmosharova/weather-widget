This is a weather widget to display local weather
It will be used on a monitor that is on display in a hallway

The widget should be data driven, showing the data in a useful and information dense way

weather should be for Berlin, Germany

Data to show:
Current temperature
a graph of temperature over the next three days, showing highs and lows directly on the graph
a picture indicating current weather
precipitation chance and millimeters
cloud coverage

Take inspiration from Deutsche Wetter Dienst, Darksky, and other data focused weather apps.

It should update every 10 minutes or upon request
Use the BrightSky API to retrieve data
